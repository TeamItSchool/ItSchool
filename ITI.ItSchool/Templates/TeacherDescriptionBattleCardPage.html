Description

Règle du jeu :
L'enfant possède une main de 5 cartes. Parmi ces cartes, il doit cliquer sur la ou les cartes correspondant à la carte modèle située à droite.

Temps :
L'enfant a 1 minute pour réaliser l'exercise.

Personnalisation :
Vous pouvez choisir entre 3 modes de difficultés. Le mode facile sera affecté à toute la classe et vous aurez la possibilité de choisir les élèves pour les deux autres niveaux.

Ensuite vous avez le choix entre l'alphabet et les différentes catégories de nombres proposées.
